# Description
The Drawing App is a versatile web application that allows users to express their creativity through digital drawing. Built with JavaScript, this app provides a canvas for users to sketch, draw, and paint with various tools and features.

# Features
Brush, pencil, and eraser tools for drawing.

Color palette with a wide range of color options.

Undo and redo functionality for easy corrections.

# Installation
To use the Drawing App locally, follow these steps:

1.Clone the repository: git clone https://github.com/RadinaAvramova/drawing-app.git

2.Navigate to the project directory:
cd drawing-app

3.Open the index.html file in your web browser.

# Usage
Once the application is opened in your web browser, you can start drawing on the canvas using the available tools and features. Customize your drawing with various colors and brush sizes.

# Controls
Mouse: Interact with the drawing canvas.

Brush Tool: Select the brush tool for freehand drawing.

Pencil Tool: Select the pencil tool for precise lines.

Eraser Tool: Select the eraser tool to erase parts of your drawing.
